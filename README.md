# 后湖智慧科创区控制性详细规划与城市设计

> 长沙市岳麓区后湖片区 | 2022级城乡规划专业课程设计
>
> **核心命题**：存量时代下，大学城边缘的"消极空间"如何通过城市更新转型为景城产人融合的科创街区？

---

## 项目概览

本课题以长沙市岳麓区后湖片区约20公顷地块为对象，完成从**上位规划解读 → 现状综合调研 → 城市设计方案 → 控制性详细规划转化**的全流程规划实践。项目涵盖政策分析、空间数据采集与可视化、建筑功能策划、城市设计、控规图则编制等环节。

**"青创绿链 · 后湖客厅"** —— 以绿为链，产城融合，全龄共享。

### 关键指标

| 指标 | 数值 | 约束来源 |
|------|------|----------|
| 用地面积 | ~20 ha | 课程任务书划定 |
| 建筑栋数 | 34 栋 | 方案设计 |
| 建筑密度 | 25% | 控规上限 28%（长沙技规2024） |
| 容积率 | 0.35 | 控规上限 1.2 |
| 绿地率 | ≥30% | 岳麓山风景名胜区管控 |
| 建筑限高 | 14-18 m | 岳麓山风景名胜区管控 |
| 公共空间占比 | 75% | 方案设计 |

## 技术栈与工作流

### 地理空间数据处理

- **CAD** (AutoCAD 2020): 地形底图清理、图层标准化、标高统一、多地块边界对接
- **GIS** (ArcGIS/QGIS): 用地分类、服务半径分析、地形坡度分析
- **数据可视化**: 基于 Leaflet 的交互式 HTML 地图，实现 735 栋建筑的分层设色渲染与点击详情查询、253 个 POI 的多类别切换展示与服务半径叠加

### 三维建模

- **SketchUp**: 全要素三维场景 —— 地形、水系、道路、广场、绿地、34栋建筑单体、行道树、景观节点，含标准分层标签体系
- **AI 辅助渲染**: 基于 GPT Image 2 的规划效果图生成管线，从 SU 线稿截图到专业级人视点效果图

### 数据分析与自动化

- **Python**: 建筑功能自动策划脚本（`gen_building_manual.py`），批量生成标准化文档
- **JSON**: 建筑属性数据（坐标/高度/层数/分类）与 POI 数据的结构化存储，可直接导入 GIS

### 文档与表达

- **Microsoft Office**: 完整的规划文本、控规说明书、经济技术指标表
- **Adobe 系列**: A1 展板排版、分析图后期
- **Markdown**: 全流程项目文档、分析图提示词库

## 核心能力展示

### 1. 上位规划解读与政策分析

系统梳理了 12 份从中央到长沙市级的政策文件和上位规划，提取关键约束条件并建立分析框架：

| 层级 | 代表文件 | 核心约束 |
|------|----------|----------|
| 国家 | 《城市更新行动意见》(2025)、自然资源部 226号文 | "留改拆"模式、工改用地政策 |
| 省级 | 湖南省城市更新实施意见(2025)、长株潭都市圈规划(2022) | 历史文化保护、"长沙研发"定位 |
| 市级 | 长沙国土空间总规(2021-2035)、岳麓山风景名胜区规划 | **限高14-18m、容积率≤1.5、绿地率≥30%** |

**能力关键词**: 政策解读 | 多层级约束提取 | 上位规划衔接

### 2. 现状综合调研与空间数据可视化

- 对 735 栋现状建筑逐一分类（保留/改造/拆除），生成交互式分类地图
- 采集并可视化 253 个配套设施 POI（教育23/医疗6/商业61/交通145/文化18）
- 编制四段式历史文化沿革（古代书院 → 工业仓储 → 城中村转型 → 创意高地）
- 产业现状与区域联动分析（45%文创主导）

**能力关键词**: 田野调查 | 数据可视化 | 多源数据融合 | 现状诊断

### 3. 城市设计方案

以"青创绿链·后湖客厅"为概念，在严格限高（14-18m）和低密度（容积率≤1.2）约束下完成：

- **功能分区**: 艺术教育区(1.9ha) + 科创中心(4.0ha) + 成果实践区(3.6ha)
- **道路系统**: 4m外围环路 + 2.5m内部支路 + 6m中央绿轴 + 3m环湖绿道
- **公共空间体系**: 核心广场(3个) → 线性绿道(2条) → 内部庭院(9个) → 口袋公园(5个)
- **建筑布局**: 34栋低层院落式，2-3层为主，层高3.5m

**能力关键词**: 概念生成 | 空间结构 | 约束下的创新 | 人本尺度设计

### 4. 控制性详细规划转化

将城市设计方案"翻译"为控规语言：
- 按 GB 50137-2011 标准进行用地分类与代码标注
- 编制道路工程、公共服务设施、市政设施专项规划图
- 典型地块控规分图图则（指标控制 + 城市设计导引）
- 用地平衡表 + 分地块经济技术指标

**能力关键词**: 规范应用 | 指标管控 | 从方案到管制

### 5. 技术工具链搭建

- `gen_building_manual.py`: 自动化生成 34 栋建筑的逐栋功能定义手册，含功能定位、空间场景、设施配置、流线衔接四维度。体现了**用编程解决重复性规划工作**的意识
- `分析图提示词库.md`: 沉淀了 6 组经过验证的 AI 辅助出图提示词模板（建筑高度控制、天际线、绿地系统、公共空间、节点效果图等），可复用、可迭代

**能力关键词**: Python自动化 | AI工具链 | 工作流沉淀 | 可复用方法论
