## 4. 公共空间规划图

**底图来源**：CAD 导出建筑轮廓 + 道路边线 + 水岸线（纯线稿，白底黑线）

```
A professional Chinese urban planning analysis diagram for "公共空间规划图" (Public Open Space Plan). Planning institute competition board style. This diagram is about SPATIAL HIERARCHY and PEDESTRIAN ACCESSIBILITY, not about green coverage.

COMPOSITION: Main plan (full canvas) + 3 overlay panels.

MAIN PLAN (full canvas):
- 34 building footprints in light grey (#E8E8E8), with thin dark grey outlines. Buildings are background, not the focus.
- Lake in soft blue-grey wash. Roads in thin light grey.
- FOUR-LEVEL PUBLIC SPACE HIERARCHY as the dominant visual layer:
  LEVEL 1 "核心广场" (3 spaces: 入口广场、中央广场、滨湖广场): deep terracotta red #C44D34, LARGE outlined circles or polygons with bold hatching fill. Each labeled with name and area in white text on red background. These are the most visually dominant elements.
  LEVEL 2 "线性绿道" (2 corridors: 中央绿轴、环湖绿道): continuous bands in warm orange-brown #D4956A, 8-10mm stroke width. Labeled along the route. Add small arrowheads showing walking direction.
  LEVEL 3 "内部庭院" (9 spaces within building clusters): medium ochre-yellow #E0C080, filled rectangles with thin borders. Small labels.
  LEVEL 4 "口袋公园" (5 small scattered spaces): light cream-yellow #F0DFB0 with dotted borders. Minimal labels.
- SERVICE RADIUS: From each Level 1 space, draw dashed concentric circles at 300m radius (thicker dash, labeled "300m步行可达（5分钟）") and 500m radius (thinner dash, labeled "500m步行可达（8分钟）"). Use muted blue-grey for these circles so they don't compete with the hierarchy colors.
- PEDESTRIAN NETWORK: Thin dashed orange lines connecting ALL spaces across all four levels into one continuous network. Add small white circle nodes at each connection point.
- ENTRANCE/ACCESS POINTS: Small black triangles along the site boundary marking public entry points, labeled "入口".

OVERLAY PANEL 1 (BOTTOM-LEFT, ~20% of canvas):
- Schematic axonometric or exploded diagram showing the four levels stacking vertically: 口袋公园 (bottom, widest) → 内部庭院 → 线性绿道 → 核心广场 (top, most prominent)
- Annotate each level with quantity, typical size, and function

OVERLAY PANEL 2 (BOTTOM-RIGHT, ~15% of canvas):
- Cross-section through a typical public space sequence: 核心广场 | 线性绿道 | 内部庭院 | 口袋公园 → showing spatial enclosure and scale
- Human figures for scale reference at each space type

OVERLAY PANEL 3 (TOP-RIGHT, ~15% of canvas):
- PROMINENT data card with key metric: "公共空间占比 75%" in large bold numbers
- Below: "核心广场：3处 / 线性绿道：2条 / 内部庭院：9处 / 口袋公园：5处"
- "300m步行覆盖率：XX%"

LEGEND: Top-left. Title "公共空间规划图" bold 黑体. Four color swatches with hierarchy labels.

COLOR PALETTE: Warm terracotta-to-cream gradient for the four levels. Buildings and lake are muted background. Background: warm off-white #FAF8F5. NO green as the primary color — this diagram is about SPACES, not vegetation.

All Chinese labels. Scale bar + north arrow bottom-right.
```

## 对比说明：绿地系统 vs 公共空间

| | 绿地系统分析图 | 公共空间规划图 |
|---|---|---|
| **分析对象** | 绿化的"量"和"网" | 空间的"级"和"序" |
| **主色调** | 绿色系 | 暖土色系 |
| **核心指标** | 绿地率30%、绿道总长 | 公共空间占比75%、步行覆盖率 |
| **表达方式** | 绿地分类填色 + 饼图 + 绿廊连接 | 四级体系 + 服务半径圈 + 步行网络 |
| **评审关注点** | 生态性、连续性 | 公共性、可达性 |
